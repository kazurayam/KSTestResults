Package: com.kazurayam.materials

[Groovy] Interface MaterialStorage

  • MaterialStorage is an external directory for Materials outside a Katalon Studio project. You can backup your Materials from the project's Materials directory to the MaterialStorage. Also you can restore a version of TSuiteExecution

      • long reduce(long targetBytes)

        Calcute the total file size in the Storage to check if it exceeds the target size in bytes. All of TSuiteNames are inspected as possible deletion target. If exceeding, clear older TSuiteResults to reduce the total less than the target. e.g., MaterialStorage.reduceSizeTo(20 * 1000 * 1000 * 1000); // 20 giga-bytes

        Returns:
        the size of the Storage after reduction = sum of remaining Material files
        Parameters:
        targetBytes - want to make the Storage size less than this
